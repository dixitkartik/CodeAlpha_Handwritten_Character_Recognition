# ✍️ Handwritten Character Recognition using Deep Learning

## CodeAlpha Machine Learning Internship - Task 3

### Objective

The objective of this project is to develop a Deep Learning model capable of recognizing handwritten digits from images using the MNIST dataset. The model is trained using TensorFlow/Keras and can accurately classify handwritten digits from 0 to 9.

---

## Technologies Used

- Python
- Google Colab
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Scikit-learn

---

## Dataset

- MNIST Handwritten Digits Dataset
- 70,000 grayscale images
- Image Size: 28 × 28 pixels
- Classes: 10 (Digits 0–9)

---

## Workflow

1. Import Libraries
2. Load MNIST Dataset
3. Explore Dataset
4. Data Preprocessing
5. Build Neural Network
6. Compile Model
7. Train Model
8. Evaluate Model
9. Make Predictions
10. Save Model

---

## Results

The trained model achieved high accuracy (around 97–99%) on the test dataset and successfully recognized handwritten digits.
